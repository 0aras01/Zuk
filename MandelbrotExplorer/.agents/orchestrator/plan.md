# Project Plan

This document outlines the steps to refactor the Fractal Explorer application.

## Steps
1. **Exploration**: Analyze the codebase, existing tests, and architecture.
2. **Setup Test Track**: Establish the E2E testing framework to ensure no regressions.
3. **Decomposition**: Define architecture/sub-ViewModels boundaries.
4. **Implementation**:
   - Milestone 1: NavigationViewModel
   - Milestone 2: DiagnosticsViewModel
   - Milestone 3: RenderingViewModel
   - Milestone 4: MainViewModel coordination and XAML integration
   - Milestone 5: Logging integration
5. **Verification**: Run all tests and verify criteria (MainViewModel under 300 lines, compiles, all features work).
